# KilaFlow-Mods
a list of mods i've created for KilaFlow

To view and download from the list of mods, click on the "mods" folder above, and you should see folders for each mod. Click on one, then, you should see a README explaining the mod, any configurations for the mod, and the actual mod download itself, being the zip file. To download the mod, download the zip file and drag them into your `mods` directory while making sure you have the KilaFlow Mod Loader installed from here: https://github.com/Landiie/KilaFlow-Mod-Loader
